## Movie sources

From Consumet API  
You can refer to [here](https://github.com/consumet/api.consumet.org)

## Main technology used

- Nextjs, React, Typescript, Tailwind
- Prisma
- MongoDB
- Swiper (slider)
- Next-Auth
- react-infinite-scroll-component (Infinite loading)
